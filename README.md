# PPDB Online Sekolah
Aplikasi Pendaftaran Siswa Baru 


Menggunakan :
- Template SB-Admin 2
- Bootstrap 4
- PHP 7 NATIVE
- MySQL
- Database di folder "db/pendaftaran_siswa.sql"

Pengguna :
- User : Admin, Siswa

Lingkup Aplikasi :
- Landing Page Pendaftaran
- registrasi Siswa
- Dashboard Siswa dan Admin
- Edit Profil Siswa
- Login Sistem
- Rekap Data Pendaftar
- Cetak PDF
